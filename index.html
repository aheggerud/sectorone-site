How it works

Local agent in the garage + secure cloud ‚Äî built for reliability first.

1) Ingest & Normalize

Watch folders on the garage laptop ingest i2/CSV/VBO/video. We enforce units, align clocks, and map to a canonical event schema: session key, car, driver, compound, weather, timestamp.

2) Store & Version

Time-series DB for channels, S3-class object storage for binaries, Postgres for metadata. Git-like versioning for setups/workbooks with human-readable diffs.

3) Automate & Share

Serverless jobs generate debrief packs; cross-media search indexes let you retrieve moments in context. RBAC & audit make sharing safe.
